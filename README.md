# sjparkinson/dotfiles

Personal fork of [cowboy/dotfiles][1].

[1]: https://github.com/cowboy/dotfiles

## Usage

```bash
$ bash -c "$(curl -fsSL https://raw.github.com/sjparkinson/dotfiles/master/bin/dotfiles)" && source ~/.bashrc
```
